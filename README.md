# Coders of Delhi - Social Network Data Analysis

## Description

This project analyzes a social network dataset using Python. It demonstrates how to load, clean, and analyze user data while implementing a simple "People You May Know" recommendation system.

## Project Tasks

### Task 1 - Data Loading
- Load JSON data
- Display users and pages

### Task 2 - Data Cleaning
- Remove users with empty names
- Remove duplicate friends
- Remove inactive users
- Remove duplicate pages
- Save cleaned data

### Task 3 - People You May Know
- Find friend recommendations using mutual friends
- Return suggested users with mutual friend counts

## Technologies Used
- Python
- JSON
- Jupyter Notebook

## Files
- Data_load.ipynb
- data_cleaning.ipynb
- known.ipynb
- file.json
- betterdata.json

## How to Run
1. Clone the repository.
2. Open the notebooks in Jupyter Notebook or JupyterLab.
3. Run the notebooks in order:
   - Data_load.ipynb
   - data_cleaning.ipynb
   - known.ipynb
